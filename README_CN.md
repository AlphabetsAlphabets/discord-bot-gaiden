我决定创造为了获得经验。加上我大约一年前尝试创造它时，我的Python非常差。

# 特征
1. 反应搜索功能
Gaiden可以对用户的图是反应. 你可以翻到第二页，和第一页之间。不需要使用时，你可以加一个复选标记在信息，Gaiden就会吧“搜索信息”删除。这样会避免屏幕混乱。
